# TP-JAVA


### Estructura del proyecto

```
main/src/com/      
 ├── servicio/       # Lógica de Managers y CRUD (GestorClientes, GestorPedidos, GestorProductos)
 ├── modelo/         # Modelos (Cliente, Pedido, Producto y sus subclases)
 ├── excepciones/    # Clases de Excepciones Personalizadas
 ├── ui/             # Interfaz de Usuario (MenuPrincipal y sus Helpers)
 └── Main.java       # Punto de entrada
```
